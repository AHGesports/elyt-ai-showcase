# Responsible Use

Elyt is intended for legitimate browser automation where the user owns the workflow, controls the accounts involved, or has clear authorization to automate the target process.

Recommended uses include:

- Internal operations.
- QA and regression workflows.
- Authorized data collection.
- Product and price monitoring where permitted.
- Repetitive admin workflows.
- Multi-account workflows for accounts the operator owns or is authorized to manage.

Do not use Elyt or material from this repository for:

- Credential theft or account compromise.
- Spam, harassment, or deceptive outreach.
- Circumventing access controls.
- Violating third-party terms, law, or platform policy.
- Scraping private or restricted data without authorization.
- Fraud, scalping, or other abusive automation.

This public showcase intentionally excludes private implementation details, source code, credentials, provider internals, and operational bypass instructions.
